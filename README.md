# Java ☕

Bem-vindo ao meu repositório central de Java! Este é um espaço versátil onde documento minha evolução na linguagem, reunindo desde implementações profissionais até exercícios acadêmicos e testes de novas tecnologias.

## 📌 O que você encontra aqui?

* **Projetos Acadêmicos:** Implementações de algoritmos, estruturas de dados e trabalhos de faculdade/cursos.
* **Laboratório Profissional:** Mockups de sistemas, implementações de Design Patterns e boas práticas de mercado (Clean Code, SOLID).
* **Snippets & Utilidades:** Pequenos códigos úteis para o dia a dia.

## 🛠️ Como rodar os projetos

Como este repositório contém múltiplos subprojetos, recomendo:
1. Clonar o repositório
2. Navegar até a pasta específica do projeto que deseja explorar.
3. Verificar o arquivo da pasta para entender as dependências.
